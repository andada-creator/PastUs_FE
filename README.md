# PastUs_FE
React Native(Expo) 기반 모바일 앱 학습 프로젝트

## Tech Stack
- React Native
- Expo
- JavaScript

## 실행 방법
npm install
npx expo start

본 프로젝트는 Spring Boot 기반 백엔드 서버와 연동하여 동작하도록 설계되었습니다.  
현재 저장소에는 프론트엔드 코드만 포함되어 있으며, 백엔드 서버가 실행되지 않은 환경에서는 게시글 데이터가 표시되지 않을 수 있습니다.

테스트 모드를 통해 로그인 및 기본 화면 흐름은 확인할 수 있습니다.
